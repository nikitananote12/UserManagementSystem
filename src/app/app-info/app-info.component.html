User Management System

How App Works ?
  • This application is developed to showcase the listing of Employees
  • User can Add, Edit or Delete an employee
  • Employee details are listed into a table
  • App contains Sort and Filter at each column
  • App contain Pagination at the top of table

Application is having two tabs -

  1. How it Works ?   tab consist of user manual
  2. User List   tab contains Employee Management stuff

User selecting User List tab, which will navigate to Employee listing page.
This screen displays list of employee where user can add, edit or delete the employee

Add Button

For adding an employee, need to click on Add User button, this will navigates to a screen where user can update user info
Validation - All are required fields to be filled by user without that he will not able to add employee details. beside Add button app have one Cancel button, with that user can navigate back to employee listing screen

Edit Button

If user wants to update any information, he will have to select the checkbox. On checkbox selection Edit menu will appear. Onclick of that menu user will navigate to edit screen. There user can update the information and click on Update User button, this will update employee information and navigate back to employee listing If user do not wants to update and go back to listing page, He can click on Cancel button

Delete Button

If user wants to delete any record. He will have to select the checkbox, on checkbox selection Delete menu will appear and on on click of delete menu, the selected records will get deleted and the employee list will get updated

Note - Multiple records can be deleted at a time
